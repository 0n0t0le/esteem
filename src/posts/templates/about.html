eSteem - where anyone can earn rewards.

eSteem is powered by Steem platform. App is created by @good-karma and it is open-source, community driven mobile project.
eSteem offers content access for reading, commenting, voting, posting, transfer, etc. (and many more for mobile specific functionalities) features offered by Steem blockchain and Steemit.

1. App never access or hold onto user funds.
2. App offer a client-side security model, with private keys hosted locally and never sent to any servers.
3. App offers simple, attractive user interfaces and experiences
4. App never require users to input any personal information
